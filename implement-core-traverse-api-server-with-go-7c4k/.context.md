# Goal
Build the main Traverse HTTP/gRPC server in Go with all core functionality.

# Acceptance Criteria
- [ ] Go module initialized at go.mod
- [ ] HTTP REST API with all endpoints from spec/API.md
- [ ] Request manager with SQLite/PostgreSQL storage
- [ ] Authentication middleware (API key and mTLS)
- [ ] Audit logging to file/syslog/webhook
- [ ] Health check endpoint at /health
- [ ] Graceful shutdown handling
- [ ] Configuration loaded from YAML/environment

# Files to Create
- go.mod
- cmd/traverse/main.go (entry point)
- internal/server/server.go (HTTP server setup)
- internal/api/handlers.go (request handlers)
- internal/auth/middleware.go (authentication)
- internal/storage/database.go (DB abstraction)
- internal/audit/logger.go (audit logging)
- internal/config/config.go (configuration)

# Dependencies
- github.com/gin-gonic/gin (HTTP framework)
- gorm.io/gorm (ORM)
- gorm.io/driver/sqlite
- gorm.io/driver/postgres
- github.com/golang-jwt/jwt/v5 (JWT tokens)
- github.com/spf13/viper (config management)
