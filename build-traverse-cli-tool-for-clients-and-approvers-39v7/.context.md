# Goal
Create CLI tool for requesting secrets, checking status, and approving requests.

# Acceptance Criteria
- [ ] traverse CLI binary with subcommands
- [ ] `traverse request` - submit secret request
- [ ] `traverse status` - check request status
- [ ] `traverse approve` - approve pending requests
- [ ] `traverse deny` - deny pending requests
- [ ] `traverse get` - retrieve secret with token
- [ ] `traverse server` - start the server
- [ ] Interactive mode with real-time updates

# Files to Create
- cmd/traverse/cli/main.go (CLI entry)
- cmd/traverse/cli/request.go (request command)
- cmd/traverse/cli/approve.go (approve/deny commands)
- cmd/traverse/cli/get.go (get secret command)
- cmd/traverse/cli/status.go (status command)
- pkg/client/client.go (API client library)

# Key Implementation Details
- Use Cobra for CLI framework
- Support for config file (~/.traverserc)
- Polling mode with progress indicator
- Environment variable for API key: TRAVERSE_API_KEY
- Tab completion support
